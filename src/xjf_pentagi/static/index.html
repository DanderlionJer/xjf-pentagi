新手引导：不懂渗透测试也可以按这个顺序做

  1. 第 1 步 合规：只对你有书面授权或自有资产测试。未授权扫描可能违法。
  2. 第 2 步 填写测试目标：左侧「测试目标」输入域名或完整 URL。
  3. 第 3 步（推荐新手） 勾选「仅校验/干跑」：先不真实执行工具，熟悉流程。
  4. 第 4 步 选模式：上方三个标签——「测试模块」单步点工具；「固定流程」跑预置流水线；「全链路自动化」一键连贯。
  5. 第 5 步（可选） 大模型：若要智能规划，勾选「启用 LLM」并填 API 或配置环境变量。
  6. 第 6 步 点击对应按钮执行，在页面下方「输出」查看结果与报错。

提示：必须用浏览器打开 http://127.0.0.1:8080 （不要用双击 index.html 的 file:// 方式）。若设置了 XJF_UI_TOKEN，左侧「UI 访问令牌」须填一致。
